
# CasinoPlus

A lightweight casino overhaul mod for Schedule I [alternate-mono].






## Features

- Blackjack Double Down and Split
- Casino open 24/7
- Slots betting ranges $100 - $5000
- Table games betting ranges $1K - $25K
